Privacy Policy for Battery Percentage Icon

This privacy policy outlines the practices for the Battery Percentage Icon app, which displays a native system tray icon to show the current battery capacity as a percentage number. The app also provides a user interface to show other battery details and includes links for users to visit the app's GitHub repository and a PayPal donation page.

Data Collection and Use

The Battery Percentage Icon app does not access or store any personal or user data. It operates solely to provide battery information and related functionalities as described.

Contact Information

If you have any questions or feedback regarding this privacy policy, you can contact us at soleon@live.com or submit feedback through our GitHub issues page.
